## 🔨 安装使用

- 获取项目代码

```bash
git clone https://gitee.com/matevip/artemis.git
```

- 安装依赖

```bash
cd artemis
yarn install
```

- 运行

```bash
yarn serve
```

- 打包

```bash
yarn build
```
